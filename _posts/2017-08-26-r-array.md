<code> array(data, dim) </code> to create a multidimensional array

```
a <- array(1:(2*2*3), c(2,2,3))

# 1st-2nd dim filled by column
a[1,1,1] # 1
a[2,1,1] # 2
a[1,2,1] # 3
a[2,2,1] # 4

# Access like a matrix
 a[,,1]

```
