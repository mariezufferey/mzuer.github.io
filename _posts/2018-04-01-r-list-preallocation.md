Lists can be pre-allocated: with <em>vector(mode = "list")</em>

```
result <- vector(mode = "list", 3)
print(result)
#> [[1]]
#> NULL
#> 
#> [[2]]
#> NULL
#> 
#> [[3]]
#> NULL


```
