```
xor(1,0) 
# [1] TRUE
xor(1,1) 
# [1] FALSE
xor(0,0)
# [1] FALSE
xor(TRUE, TRUE) 
# [1] FALSE
xor(FALSE, TRUE) 
# [1] TRUE
```
